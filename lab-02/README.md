# Лабораторная работа №2
# Исследование методов случайного поиска экстремума функции одного переменного

## Задание (вариант №6)

На интервале [-5, 2] задана унимодальная функция одного переменного
`f(x) = (1 - x)^2 + exp(x)`. Используя метод случайного поиска,
найти минимум `f(x)`. Определить необходимое число испытаний `N`.
Численный эксперимент выполнить для значений `P = 0.9..0.99`
и значений `E = (b - a)q`, где `q = 0.005, 0.01, ..., 0.1`.
При аналогичных исходных условиях осуществить поиск минимума `f(x)`,
модулированной сигналом `sin(5x)` , т.е. мультимодальной функции `f(x) * sin5(x)`.
